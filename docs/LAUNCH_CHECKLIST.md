# Launch Checklist

## Pre-Launch (Before 21.12)

### Week 1: Setup

- [ ] Set up PostHog analytics
- [ ] Configure Stripe Payment Link
- [ ] Set up Resend email
- [ ] Test email capture flow
- [ ] Test Stripe checkout flow
- [ ] Verify all routes return 200
- [ ] Set up environment variables

### Week 2: Content

- [ ] Create all social media posts
- [ ] Write cold email templates
- [ ] Prepare LinkedIn DMs
- [ ] Create community posts
- [ ] Prepare demo script
- [ ] Set up calendar for demos

### Week 3: Outreach

- [ ] Send 50+ LinkedIn DMs
- [ ] Send 25+ cold emails
- [ ] Post on social media daily
- [ ] Post on communities
- [ ] Book 5+ demos
- [ ] Follow up with leads

---

## Launch Day (21.12)

### Morning (9:00 AM)

- [ ] Verify site is live
- [ ] Test all forms and checkouts
- [ ] Post launch announcement on all channels
- [ ] Send email to waitlist
- [ ] Monitor analytics

### Midday (12:00 PM)

- [ ] Check email submissions
- [ ] Check founding customer purchases
- [ ] Reply to questions/comments
- [ ] Share social proof (if any)
- [ ] Post follow-up content

### Evening (6:00 PM)

- [ ] Review day's metrics
- [ ] Thank early customers
- [ ] Plan next day's content
- [ ] Follow up with leads

---

## Post-Launch (After 21.12)

### Week 1

- [ ] Deliver product to founding customers
- [ ] Collect feedback
- [ ] Iterate based on feedback
- [ ] Continue outreach
- [ ] Track conversions

### Week 2

- [ ] Analyze what worked
- [ ] Double down on top channels
- [ ] Improve messaging
- [ ] Scale successful tactics
- [ ] Set new targets

---

## Daily Actions

### Morning Routine (30 min)

1. Check analytics (page views, conversions)
2. Check email submissions
3. Check founding customer purchases
4. Reply to overnight messages
5. Plan day's outreach

### Midday Routine (1 hour)

1. Send 10 LinkedIn DMs
2. Send 5 cold emails
3. Post on social media
4. Reply to responses
5. Book demos if requested

### Evening Routine (30 min)

1. Review day's metrics
2. Follow up with leads
3. Plan tomorrow's content
4. Update CRM/spreadsheet
5. Set next day's targets

---

## Weekly Review

### Every Monday

1. Review last week's metrics
2. Identify top-performing channels
3. Adjust strategy if needed
4. Set week's targets
5. Plan week's content

---

## Key Metrics to Track

- **Traffic:** Page views, unique visitors
- **Engagement:** CTA clicks, email submissions
- **Conversions:** Founding customers, revenue
- **Channels:** Which sources drive most traffic
- **ROI:** Revenue vs. time spent

---

## Success Criteria

**Week 1:**
- 50+ email submissions
- 2-3 founding customers
- €300-450 revenue

**Week 2:**
- 100+ email submissions
- 5-7 founding customers
- €750-1,050 revenue

**Week 3:**
- 150+ email submissions
- 8-10 founding customers
- €1,200-1,500 revenue

---

## Red Flags

- **No traffic:** Check if site is live, verify links
- **No conversions:** Review messaging, test CTAs
- **High bounce rate:** Check page load time, review content
- **Low engagement:** Test different messaging, improve CTAs

---

## Quick Wins

1. **Personal Network:** Start with people you know
2. **Warm Introductions:** Ask mutual connections
3. **Community Engagement:** Be helpful, not salesy
4. **Social Proof:** Share early customer wins
5. **Urgency:** "Only X spots left"

---

## Tools Needed

- PostHog (analytics)
- Resend (email)
- Stripe (payments)
- Calendly (demos)
- LinkedIn (outreach)
- Twitter/X (social)
- Spreadsheet (CRM)

---

## Final Notes

- **Be Consistent:** Daily actions compound
- **Be Patient:** First sales take time
- **Be Helpful:** Answer questions, provide value
- **Be Grateful:** Thank every customer
- **Be Data-Driven:** Track everything, adjust based on data

**Remember:** The goal is conversations, not just clicks. Quality over quantity.

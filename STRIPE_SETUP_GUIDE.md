## üí° PRO TIPS:

1. **Start Simple:**
   - Use "Individual" business type
   - Use Whop store as website
   - Can always update later

2. **Statement Descriptor:**
   - Keep it short and clear
   - Customers will see this on bank statements
   - Make it professional

3. **Verification:**
   - Stripe may ask for ID verification
   - Have ID ready (passport/driver's license)
   - Usually takes 1-2 business days

4. **Test Mode:**
   - Stripe has test mode
   - Use test cards to verify everything works
   - Switch to live mode when ready

## üö® COMMON QUESTIONS:

### **"I don't have a website"**
- Use Whop store URL (you'll have one after Whop setup)
- Or use social media link temporarily
- Can update later

### **"What if I'm not a registered business?"**
- Use "Individual" business type
- Perfectly fine for starting
- Many successful businesses start this way

### **"What address should I use?"**
- Use your personal address
- Or business address if you have one
- Stripe needs it for verification

### **"What tax ID?"**
- If Individual: Use your SSN/Personal Tax ID
- If Company: Use business tax ID
- Required for tax reporting

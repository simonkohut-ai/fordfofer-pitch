## üìê Usage Guidelines

### Logo Sizing
- **Minimum Width:** 200px (logo.svg)
- **Minimum Size:** 60px (icon versions)
- **Maximum Width:** No limit (vector scales)

### Clear Space
- **Rule:** 1x the height of the unicorn icon
- **Example:** If icon is 60px, use 60px clear space

### Backgrounds
- **Light Backgrounds:** Use full color version
- **Dark Backgrounds:** Use inverted version (white text)
- **Gradient Backgrounds:** Use with shadow for contrast

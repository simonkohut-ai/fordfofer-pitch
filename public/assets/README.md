# Assets

This folder is for visual and brand assets used in the GoLoCapo / Fordfofer pitch.

Recommended contents:

- **Screenshots** of the live app (PNG/JPEG):
  - Landing page (`/`)
  - Pre‑launch page (`/prelaunch`)
  - Pricing (`/pricing`)
  - War Room (`/war-room`)
  - Client microsites (`/clients/*`)
- **Diagrams** (PNG/SVG/PDF):
  - High‑level architecture (browser → Vercel → APIs → Stripe/OpenAI/Meta).
  - Funnel flow (traffic → pre‑launch → pricing → Stripe → automation).
- **Logos / wordmarks**:
  - GoLoCapo logo.
  - Simple text lockups for slides and one‑pager.

> Tip: name files descriptively, e.g. `landing-home.png`, `war-room.png`, `architecture-diagram.svg`, `logo-golocapo-dark.svg`.
